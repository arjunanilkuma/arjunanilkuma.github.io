# Project Overview
I designed a two-stage gearbox for an electric go-kart using **MATLAB** for optimization and **Siemens NX** for CAD.

### Key Specs:
* **Reduction Ratio:** 4:1
* **Material:** 7075-T6 Aluminum
* **Simulation:** FEA conducted in Abaqus
